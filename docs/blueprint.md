# **App Name**: EyeFortress Parallax

## Core Features:

- Configurable System Parameters: Allow users to change system name, description, and WebP parallax image sequences.
- Parallax Hero Section: Implement a full-screen hero section with a WebP image sequence as the background, controlled by scrolling.
- Dynamic Content Update: Update the hero overlay text, theme color, and WebP sequence when switching CCTV solutions with fade in fade out transition.
- Loading Experience: Display a loading overlay with a progress bar before showing the hero and a loading indicator when switching solutions.
- Section Navigation: Create a sticky top navigation bar with links to main sections, indicating the active section while scrolling.
- Theme Customization: Enable theme customization of different sections to ensure uniformity throughout the website.
- CCTV Solutions Management: Admin UI to maintain an array of CCTV solutions, each with a name, subtitle, description, theme color, mode, and WebP frame sequence.

## Style Guidelines:

- Primary color: Near-black background (#121212) for a cinematic look.
- Secondary color: Bright white text (#FFFFFF) for contrast and readability.
- Accent color: Light gray (#D3D3D3) for CTAs and subtle highlights; use lighter hues for active states or toggles.
- Headline font: 'Space Grotesk' (sans-serif) for a bold, uppercase system name; body font: 'Inter' (sans-serif) for the descriptive paragraph; Note: currently only Google Fonts are supported.
- Use minimal, monochrome social icons in the footer.
- Full-screen hero section with a WebP image sequence, overlay text block on the left, variant navigation on the right, and social icons at the bottom.
- Smooth parallax scrolling with scroll-controlled WebP animation; fade-in/out transitions when switching solutions; horizontal loading bar.
